# JLPT N1 学习计划 · Phase 1

日文学习计划网页，包含 Phase 1（基础重建）的每日学习表格。

## 功能

- 每日学习任务表格（6天/周，周一至周六）
- 点击单元格标记完成，进度自动统计
- 当日列自动高亮
- 进度保存于浏览器本地
- 可折叠的「学习方法」参考

## 使用

直接访问：[https://wayne06.github.io/nihongo-plan/](https://wayne06.github.io/nihongo-plan/)

## 计划结构

- Phase 1（6/29 ~ 8/15）：绿皮书N3 + 新完全掌握N3 + N2真题
- 每日5模块：词汇 / 语法 / 听力 / 阅读 / 会话
